# Headers - Cabeçalhos
> Os "headings" são como h no html, mas no "markdown" utilizamos #.

> Veja como fazer:

#### No html
```html
<h1> h1 </h1>
<h2> h2 </h2>
<h3> h3 </h3>
<h4> h4 </h4>
<h5> h5 </h5>
<h6> h6 </h6>
```

#### No "markdown"

```markdown
  # Cabeçalho
  ## Cabeçalho
  ### Cabeçalho
  #### Cabeçalho
  ##### Cabeçalho
  ###### Cabeçalho
```
